      Conventions

      一、追求的写博目标

      简单易读、准确、有价值。内容为王。

      二、博主的话

      1、博客内容仅代表写作者写作当时的想法，随着经验的增加，其正确性和时效性，限于博主本人的能力无法保证，但本人会不断更新自己的水平，发现错误也会及时更正。

      文章力争深入浅出，不过这对于不同水平的人有不同的理解。都以我写作当时的水平和理解能力为参考线。

      2、为了维护 Blog 页面的简洁性，本人暂时没有增加评论的想法，也不会投放广告。

      3、如果有人发现文章的错误，或者想与我联系，还请通过这里列出的方式联系本人： Contact Me 。 最好附上简明正确的做法，本人将不胜感激。

      4、为了获得最佳访问体验，推荐使用 Chrome 或者 Safari 浏览器。

      三、许可协议

      本博客所有文章采用 CC BY-NC-ND 3.0 许可协议进行许可。(署名-非商业使用-禁止演绎-自由转载 )

      四、博客文章遵守的公约 ( Beta )

      1、正文字体：Georgia

      2、正文中使用 "注意"、"说明" 的格式：加粗、##### 或 <h5>

      3、摘要只要一句话、导航只用面包屑。

      4、常用的几种用于指示和提醒的 Font Awesome图标 ( 能用字符图标代替的就用字符图标代替 )：

      • 用于说明一些东西： <i class="fa fa-hand-o-right"></i>

      • 用于提出一些思路： <i class="fa fa-question"></i>

      • 用于一些不错的东西、思想： <i class="fa fa-start"></i>

      • 用于文章中提供的下载链接： <i class="fa fa-download"></i>

      • 用于提醒和注意事项： <i class="fa fa-exclamation"></i>

      5、次序：

      • 中文数字 ( 一、二、... ) 用于 <h2>、<h3>或者文章正文中 相对 二等标题，可嵌套。
        其后跟中文逗号 、。

      • 罗马数字 1、2、3、... 用于中文数字所表示标题的下一级标题。
        其后跟中文逗号 、。

      • 斜体只用于其他文章或者外部资源的引用。

      6、其他：

      • 段落长度：文章每个段落长度不要超过编辑器最大宽度，否则分段。

      • 括号：
        ①汉字和英文、符号、数字之间要有空格；
        ②括号和引号不要使用汉字符号；
        ③英语括号的左右都要使用空格隔开。

      • Tag 数目：每篇文章的 Tag 数目不要超过 5 个，如果实在会，那就多写几篇文章。
